⚙️ API Key Setup

To enable real URL checking, add your free Google Safe Browsing API key:

1. Go to console.cloud.google.com
2. Create project → APIs & Services → Library → Search "Safe Browsing" → Enable
3. APIs & Services → Credentials → Create → API Key → Copy
4. Open url_checker.py line 20 → replace YOUR_API_KEY_HERE with your key
5. Restart the server: python main.py
